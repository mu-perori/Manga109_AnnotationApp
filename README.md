# Manga109_AnnotationApp
このプロジェクトは漫画の読み順をアノテーションするアプリです。

本アプリは科学研究費助成事業 基盤研究(B) （課題番号 21H03491）
「巨大知識グラフに対するクエリ検索の近似的な高速化」
の助成を得て作成したものです。

## 動作環境
次の環境で動作することを確認しています。
- OS: macOS 10.15.7
- ブラウザ: Google Chrome 97.0.4692.99

## フォルダについて
`sigcc`: 第6回コミック工学研究会発表会にて公開したバージョンのツール

## 使い方
1. このプロジェクトをダウンロードします。
